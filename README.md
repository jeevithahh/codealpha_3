# Handwritten Character Recognition

## Overview

This project recognizes handwritten digits using Machine Learning. It uses the built-in Digits dataset from Scikit-learn and trains a Random Forest Classifier to identify handwritten numbers.

## Features

- Loads handwritten digit dataset
- Splits data into training and testing sets
- Trains a Random Forest Classifier
- Predicts handwritten digits
- Displays sample handwritten images
- Evaluates model performance
- Saves trained model using Joblib

## Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Joblib

## Dataset

Scikit-learn Digits Dataset

- 1797 handwritten digit images
- Image size: 8 × 8 pixels
- Digits: 0–9

## Machine Learning Algorithm

Random Forest Classifier

## Evaluation Metrics

- Accuracy Score
- Precision
- Recall
- F1-Score
